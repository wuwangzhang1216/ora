How it works

Four stages. One GPU. Zero network.

Stage 1
Mic
AVAudioEngine captures 16 kHz PCM.
Stage 2
VAD
Silero endpointing with hysteresis.
Stage 3
ASR
On-device speech recognition, Metal GPU.
Stage 4
Translator
On-device LLM, streaming tokens to the caption card.

All four stages run via MLX Swift on Apple Silicon's Metal GPU. Partial results stream back while you're still speaking; the final translation commits when a short silence is detected.
